# Glossary

Key terms and concepts for understanding Primus.

---

## A

### Attestation
A verified statement about data that can be shared on-chain or off-chain. Attestations are cryptographically signed to prove their authenticity.

### Attestor
A node in the Primus network that participates in the zkTLS verification process, helping to verify data authenticity.

---

## C

### Caller
An individual or organization that uses Primus network's computation capabilities and data resources. Callers initiate computation tasks and pay fees.

### Chain-Agnostic
The ability to work across multiple blockchains without being tied to a specific one. Primus is chain-agnostic, supporting Ethereum, L2s, and more.

### Core SDK
The backend SDK for server-side data verification integration.

---

## D

### Data Provider
An individual or organization that provides encrypted data to Primus network for computation. Data providers earn fees when their data is used.

### dApp
Decentralized Application. An application built on blockchain technology, often requiring smart contract interaction.

---

## F

### FHE (Fully Homomorphic Encryption)
A form of encryption that allows computation on encrypted data without decrypting it first.

---

## M

### MPC (Multi-Party Computation)
A cryptographic protocol where multiple parties jointly compute a function over their inputs while keeping those inputs private.

### MPC Mode
A zkTLS operating mode where client and attestor collaborate via MPC protocol for highest security.

---

## P

### Primus Contracts
Smart contracts deployed on multiple blockchains that manage workers, data, tasks, fees, and incentives.

### Primus Scan
A user interface for exploring Primus network information including workers, data, and tasks.

### Proxy Mode
A zkTLS operating mode where attestor forwards TLS traffic, offering better performance.

---

## W

### Worker
A node in Primus data computation network that provides computing resources, runs zkFHE algorithms, and generates proofs.

### Wallet
A digital wallet for storing cryptocurrency and interacting with blockchain applications (e.g., MetaMask).

### Whitelist
A list of approved domains allowed to use Primus SDK in production environment.

---

## Z

### zkFHE (Zero-Knowledge Fully Homomorphic Encryption)
Combination of zero-knowledge proofs and FHE, enabling verifiable computation on encrypted data.

### zkTLS (Zero-Knowledge TLS)
A cryptographic technology that verifies TLS data authenticity while preserving privacy, without requiring data source server modifications.

### zk-SNARK
Zero-Knowledge Succinct Non-Interactive Argument of Knowledge. A type of zero-knowledge proof.

### ZKP (Zero-Knowledge Proof)
A cryptographic method that allows one party to prove they know a value without revealing the value itself.
